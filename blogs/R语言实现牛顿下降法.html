R 语言实现牛顿下降法

凸是一个很好的性质.如果已经证明了某个问题是凸的,那这个问题基本上算是解决了.
最近在解决一个多目标优化的问题.多目标的问题往往是非凸的.好在能够知道这个问题的近似解大概是多少.这样这个多目标优化的问题至少能够在局部运用凸优化的方法来解决了.解决凸优化的方法有很多,比如梯度下降法,内点法.在梯度下降法中,牛顿下降法是一种重要的方法,也容易实现.更好的是牛顿下降法的收敛速度是二次的,比通常的下降法的收敛速度要快很多.

牛顿算法

从这里可以看出,如果hessian矩阵是奇异的,那么牛顿下降法将会失效.这是后就需要运用其他的算法了.比如拟牛顿法.

R语言实现(代码)

  2. newton <- function(func = objfun, x0, tol = 1e-5, n.max = 100,...){
  3.     x <- x0
  4.     g <- grad(func, x, ...)
  5.     h <- hessian(func, x, ...)
  7.     n <- 0
  8.     while( max(abs(g))>tol && n<n.max ){
  9.         x <- x-solve(h,g)
  10.         g <- grad(func, x, ...)
  11.         h <- hessian(func, x, ...)
  12.         n <- n+1
  13.     }
  14.     if(n == n.max){
  15.         cat('newton failed to converge\n')
  16.         return(x)
  17.     }
  18.     return(x)
  19. }

依赖包

numDeriv
如果需要安装,在R控制台里键入:

  1. install.package(numDeriv)

说明

func : 目标函数.
x0: 目标函数的极小化初始值.
tol:控制精度,越接近零越精确,代表梯度已经是0.
n.max:迭代次数.
... : 目标函数的其他参数.

例子

  1. testfun <- function(x, a,b){
  2.     y <- a*x[1]^2 + b*x[2]^2
  3.     return (y)
  4. }
  5. library(numDeriv)
  6. y <- newton(func = testfun, x0=c(1,1), a = 1,b = 1)
